Evaluador de Hojas de Vida ANEIAP

¿Qué tan listo estás para asumir un cargo de Junta Directiva Capitular? ¡Descúbrelo aquí! 🦁

Esta herramienta analiza el contenido de la hoja de vida ANEIAP, comparándola con las funciones y perfil del cargo al que aspiras. Evaluamos aspectos puntuales en los cuales se hace necesario que el aspirante enfatice para asegurar que se encuentre preparado.

La herramienta fue diseñada para apoyar en el proceso de convocatoria a los evaluadores para calificar las hojas de vida de los aspirantes.

Como resultado de este análisis, generamos un reporte PDF descargable con toda la información.

🔍 Selecciona el tipo de evaluación de Hoja de Vida

Versión Simplificada

Esta versión analiza la hoja de vida de forma mucho más rápida evaluando cada una de las experiencias como listado, sin descripciones detalladas.

Recomendaciones
  • Es preferible que la HV no haya sido cambiada de formato varias veces
  • La EXPERIENCIA EN ANEIAP debe estar enumerada para facilitar el análisis
  • El análisis puede presentar inconsistencias si la HV no está debidamente separada en subtítulos
Ir a Evaluador Simplificado

Versión Descriptiva

Esta versión es más cercana al entorno profesional, permitiendo analizar la descripción detallada de cada una de las experiencias de la hoja de vida.

Recomendaciones
  • Organiza tu HV en formato descriptivo para cada cargo o proyecto
  • Utiliza negrita para identificar la experiencia
  • Usa guiones para detallar las acciones realizadas en cada ítem
  • Evita usar tablas para la sección de experiencia
